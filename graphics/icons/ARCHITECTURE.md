# Icons

- `trainmode-universal.png` — полноразмерный прозрачный исходник.
- `trainmode-universal-256.png` — игровой icon asset.

```text
prototypes/entities.lua
prototypes/items.lua
prototypes/technologies.lua
  -> __TRAINMODE__/graphics/icons/trainmode-universal-256.png
```

При добавлении отдельных иконок сохранять PNG с alpha и явно задавать
`icon_size`.
