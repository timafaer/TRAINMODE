# Core

Каталог зарезервирован для общих чистых доменных функций, не относящихся только
к routing, registry или dispatcher.

Сейчас активного кода здесь нет. Новый код допустим только если:

```text
несколько runtime-слоев
  -> используют одну чистую функцию
  -> функция не обращается к Factorio API и storage
```

Алгоритм маршрутов должен оставаться в `scripts/routing/`.
